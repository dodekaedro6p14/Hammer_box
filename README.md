HAMMER BOX 

Es la compilacion de aplicaciones en HTML de creacion personal y con ayuda de IA, para mejorar sus funcionalidades, lo cree con el obejtivo de que fueran portables y cuando los necesitara, estuvieran ahi para prestar servicio. Lo mejor de todo es que son personlisados y sin publicidad y ningun tipo de suscripcion. Generalmente los utilzo por que no requieren instalacion y funcionan desde el navagador, puede que algunos te generen confictos como el Fusion_AV: que trabaja con el ffmeg de tu PC, por tanto necesitarar ejecutar un servidor el servidor.py para que no tendras problemas para utilizar cualquiera de estos script

Fusion_AV =
    Este script nace de la necesidad de unir un archivo mp4 y uno .mp3, el audio se adapta al tiempo de duracion del video, si termina el video el audio se corta, si el audio es muy corto se inicio como un bucle, sin suscripcion, publicidad y de modo oscuro, puede que tengas problemas de ejecucion, porque trabaja con el ffmeg del ordenador, por tanto si ejecutas el servidor.py no tendras problema

Lector_PDF = 

    Sencillo Lector de PDF desde el navegadro con modo oscuro y fondo animado, lo cual dejo de ser util cuando descubir el comando document.documentElement.style.filter = "invert(1) hue-rotate(180deg)";
    que invierte el color de cualquier Document o HTML que estes leyendo desde el navegador, lo ejecutas desde la consola de comandos del navegador o f12 y ya esta
